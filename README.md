# Pomotask

A simple Pomodoro-style productivity tool with categories, logs, weekly/monthly stats, goals, and countdowns.

## Quick Start

```bash
# 1) Create and activate venv
python -m venv .venv
# Windows
.\.venv\Scripts\Activate.ps1
# macOS
source .venv/bin/activate

# 2) Install deps
pip install -U pip
pip install numpy matplotlib

# 3) Run
python app.py
